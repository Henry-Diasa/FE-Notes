#### 范围

整体设计、架构设计

包括B端和编辑器、H5端、管理后台、独立组件库、自研统计服务，pv/uv统计，使用第三方统计服务（免费的）。

#### 模块设计

**模块拆分**

- B端和编辑器，比较复杂，前后端分离

- H5端，考虑性能，使用SSR，只需要服务端

- 管理后台也前后端分离，分管理前端和管理后端

- 组件库分为业务组件库和组件平台

  业务组件库渲染B端画布和H5页面，保证B端编辑器和H5内容和样式，即B端编辑器和H5的组件和渲染逻辑一致。组件平台创建、发布组件，支持业务组件库，方便组件扩展。
